## 当前状态

### ✅ 已完成的核心功能

#### 1. 基础框架
- 三界面GUI：输入 → 裁剪 → 生成
- 窗口大小：1000x800（避免UI遮挡）
- Git版本控制（已提交到main分支）

#### 2. 输入界面 (input_frame.py)
- 文件选择导入
- **拖拽导入**（使用tkinterdnd2，已修复线程安全问题）
- 路径粘贴导入
- 显示图片列表（含尺寸信息）

#### 3. 裁剪界面 (crop_frame.py) - **完整实现**
- **可拖动裁剪框**：框内拖动移动，框外拖动重绘
- **形状切换**：方形/圆形单选按钮，切换时自动刷新裁剪框和预览
- **实时预览**：150x150预览窗口，显示当前裁剪效果
- **自动刷新**：拖拽结束后自动更新预览
- **"应用并再裁一张"**：支持从同一张大图裁剪多个区域
- **圆形裁剪**：圆形外区域透明填充（使用遮罩）
- **UI布局**：右侧控制面板可滚动，避免按钮被遮挡

#### 4. 生成界面 (generate_frame.py) - **刚完成**
- **卡片格式预览**：每行4个卡片，每个卡片包含缩略图、名称输入框、尺寸信息
- **点击选中**：点击卡片切换选中状态，蓝色背景表示选中
- **全选/取消全选**：快捷按钮
- **最大组合图数**：输入框，0=无限制，必须≤选中素材数
- **"全体都有"复选框**：默认勾选，额外输出所有选中素材的组合
- **自定义名称**：每个素材可设置名称，用于生成文件名

#### 5. 排布算法 (layout_engine.py)
- **横排**：水平拼接
- **竖排**：垂直拼接  
- **圆形排布**：正多边形布局，底边水平
  - 起始角度公式：`start_angle = 90 - (180 / n)`
  - 适用于所有n≥2（已验证n=2到n=20）
  - 2个：水平排列
  - 3个：正三角形（底边2个）
  - 4个：正方形（底边2个）
  - 5个：正五边形（底边2个）
  - n个：正n边形，底边水平

#### 6. 输出功能
- 支持所有组合（单张、双张、...、n张）
- 受最大组合图数限制
- 支持"全体都有"额外输出
- 使用自定义名称生成文件名
- 进度条显示
- 完成后可打开输出目录

## 使用说明

### 运行项目
```bash
conda activate image_composer
pip install tkinterdnd2  # 如果未安装
python main.py
```

### 工作流程
1. **导入**：选择/拖拽/粘贴图片
2. **裁剪**：调整裁剪框（可拖动、可切换方形/圆形），点击"应用并再裁一张"可多次裁剪同一张图
3. **生成**：
   - 点击卡片选择要使用的素材
   - 设置名称（用于文件名）
   - 配置最大组合图数（可选）
   - 勾选/取消"全体都有"
   - 选择输出目录
   - 点击"生成输出"
